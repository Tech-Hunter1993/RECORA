RECORA is a privacy-first Android video recording app built using CameraX.  
It enables reliable video recording with Picture-in-Picture (PiP) support and a screen-lock feature to prevent accidental touches.

---

## Features
- High-quality CameraX video recording
- Picture-in-Picture (PiP) multitasking
- Screen lock mode
- Foreground service with persistent notification
- Elapsed recording timer
- Optional audio recording
- Local storage using MediaStore

---

## Privacy & Transparency
- Recording starts only by user action
- Visible preview or PiP during recording
- Persistent notification always shown
- No background or hidden recording
- No data collection or network usage

---

## Technical Details
- Language: Kotlin
- API: CameraX
- Minimum Android: 10 (API 29)
- Target Android: 14 (API 34)
